# Cómo estructurar una presentación antes de abrir Canva

## Texto para LinkedIn

Un error común al preparar una presentación es abrir Canva o PowerPoint demasiado pronto.

Antes de diseñar slides, conviene ordenar la historia.

ChatGPT puede ayudarte en esa parte: definir estructura, objetivo de cada slide, mensaje principal y cierre.

No le pidas simplemente:

"Hazme una presentación."

Mejor pídele:

"Ayúdame a estructurar una presentación de 8 slides para explicar este tema a un equipo interno. Define el objetivo de cada slide, el mensaje principal, los puntos clave y una sugerencia visual. El tono debe ser claro y profesional."

Así no partes diseñando sobre ideas sueltas.

Primero construyes el recorrido:

- contexto
- problema
- análisis
- propuesta
- próximos pasos
- cierre

Después abres Canva.

Diseñar es más fácil cuando ya sabes qué debe decir cada slide.
